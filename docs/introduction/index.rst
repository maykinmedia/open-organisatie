Introduction
============

**Open Organisatie** is a modern, open-source data- and services-layer to store
identiteit and organisatie information. Open Organisatie offers
structured data storage and services in line with the `Common Ground`_ model.

.. _`Common Ground`: https://commonground.nl/

Open Organisatie exposes several api's to store and retrieve data:

* Identiteit API
* Organisatie API

The `Notificaties API`_ is required for Open Organisatie to work if notifications are enabled, but is available as
a separate package, `Open Notificaties`_.

.. _`Notificaties API`: https://vng-realisatie.github.io/gemma-zaken/standaard/notificaties/
.. _`Open Notificaties`: https://github.com/open-zaak/open-notificaties
